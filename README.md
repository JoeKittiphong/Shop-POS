# Shop POS Offline

เว็บแอป POS สำหรับร้านเล็ก สร้างด้วย `React + Vite + IndexedDB` ทำงานออฟไลน์ในเครื่องตัวเองโดยไม่พึ่ง backend ภายนอก

## ฟีเจอร์รอบนี้

- กดเลือกสินค้าเพื่อเพิ่มเข้าบิล
- เพิ่มและแก้ไขสินค้าในเครื่อง
- คำนวณยอด รับเงิน และเงินทอน
- บันทึกประวัติการขายลง IndexedDB
- ล้างข้อมูลทั้งหมดและเริ่มใหม่ได้

## คำสั่งใช้งาน

```bash
npm install
npm run dev
```

สำหรับ build:

```bash
npm run build
```

## โครงหลัก

- `src/App.jsx` หน้า POS หลัก
- `src/db.js` จัดการ IndexedDB
- `src/App.css` และ `src/index.css` สำหรับ UI

## หมายเหตุ

- ข้อมูลทั้งหมดเก็บใน browser ของเครื่องที่เปิดใช้งาน
- ถ้าล้าง site data ของ browser ข้อมูลสินค้าและยอดขายจะหาย
- โปรเจกต์นี้ตั้งต้นให้ใช้งานออฟไลน์เป็นหลัก ยังไม่มี sync ออกภายนอก
